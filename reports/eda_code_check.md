### 카테고리별 상세 검증

#### ✅ 카테고리 1: 길이 및 구조 (Structure)
- `[1-1]`: `char_len`, `word_len` 파생 → boxplot + **평균/중간값/최대/최소** 통계 (eda.md 요구사항 완전 충족)
- `[1-2]`: `turn_count` = `\n` split 기반 발화 횟수 → boxplot + 통계 요약
- **UserWarning 해결**: `tick_params(axis='x', rotation=45)` 사용

#### ✅ 카테고리 2: 어휘 및 키워드 (Vocabulary)
- `[2-1]`: `TfidfVectorizer`로 클래스별 상위 10 키워드 추출
- `[2-2]`: **Bi-gram + Tri-gram** 모두 분석 (eda.md: "Bi/Tri-gram")
- `[2-3]`: TTR = 고유토큰/전체토큰 비율 → **boxplot**(분포) + 통계 테이블

#### ✅ 카테고리 3: 클래스 관계 (Semantic)
- `[3-1]`: TF-IDF 벡터 코사인 유사도 히트맵 + **가장 헷갈리는 클래스 쌍 순위** 출력
- `[3-2]`: 상위 50개 키워드 Jaccard 유사도 + **실제 공통 단어 목록** 출력

#### ✅ 카테고리 4: 언어적 특징 (Linguistic)
- `[4-1]`: Okt 형태소 분석기로 **명사/동사/형용사/감탄사** 4개 품사 비율 분석 (eda.md 요구 완전 충족)
- `[4-2]`: 존댓말 어미(`습니다`,`세요`...) / **반말** 어미(`해`,`야`...) / 비속어 — **3분류** 독립 분석

#### ✅ 카테고리 5: 품질 및 노이즈 (Quality)
- `[5-1]`: `!`, `?`, `ㅋㅎㅠㅜ` 패턴, `...` 빈도 분석 + 시각화
- `[5-2]`: 완전 중복(`duplicated`) + **코사인 유사도 기반 준-중복** 탐지
- `[5-3]`: **조사/기능어 18종** 클래스별 편향 히트맵 + **변동계수(CV)** 기반 편향도 정량화

#### ✅ 카테고리 6: 상황 및 맥락 (Context & Flow)
- `[6-1]`: 회사/학교/**길거리** 3개 도메인 키워드 쏠림 분석 (eda.md: "회사, 학교, 길거리")
- `[6-2]`: 대화 3등분 → **공격(위협)** + **방어(구원)** 양방향 어조 변화 (eda.md: "공수 교대 패턴")
